# 决策点

本文档描述 theo-infocard 技能中的关键决策点。

## 决策点 1: 信息密度评估

**条件**: 接收到用户内容后，需要评估信息密度

**评估维度**:
- **信息点数量**: 统计独立的信息单元数
- **文本总量**: 预估最终文本字数
- **内容复杂度**: 概念的抽象程度和关联性

**决策矩阵**:

| 信息点 | 文本量 | 复杂度 | 密度等级 |
|--------|--------|--------|----------|
| 1-2    | <100   | 低     | 低       |
| 3-5    | 100-300| 中     | 中       |
| 6+     | >300   | 高     | 高       |

**特殊处理**:
- **单一核心观点 + 详细解释** → 中密度（平衡可读性与信息量）
- **多个简单要点** → 中密度（而非高密度）
- **复杂概念 + 少量文字** → 低密度（优先视觉冲击）

## 决策点 2: 布局策略选择

**条件**: 根据信息密度选择最佳布局

**选项对比**:

### 选项 A: 大字符主义
- **适用**: 低密度内容
- **特点**:
  - 标题字号 72-140px
  - 留白占比 60%+
  - 极简装饰
  - 强烈视觉冲击
- **优势**: 高级感、记忆点强
- **劣势**: 信息承载量低

### 选项 B: 平衡布局
- **适用**: 中密度内容
- **特点**:
  - 标题字号 56-72px
  - 留白占比 40-50%
  - 清晰内容分区
  - 标准视觉层级
- **优势**: 可读性强、适用面广
- **劣势**: 冲击力较弱

### 选项 C: 多栏网格
- **适用**: 高密度内容
- **特点**:
  - 标题字号 32-56px
  - 留白占比 20-30%
  - 2-3 栏排版
  - 垂直分割线
- **优势**: 信息密度高、专业感强
- **劣势**: 易造成视觉疲劳

**决策依据**:
```
IF 密度 == "低" AND 目标 == "视觉冲击" THEN
    选择"大字符主义"
ELSE IF 密度 == "中" AND 目标 IN ["可读性", "通用"] THEN
    选择"平衡布局"
ELSE IF 密度 == "高" AND 目标 == "信息承载" THEN
    选择"多栏网格"
END IF
```

## 决策点 3: 字体选择

**条件**: 根据内容类型和语言选择字体

**字体库**:
- **Noto Serif SC**: 中文衬线，用于标题（强调、传统、权威）
- **Noto Sans SC**: 中文无衬线，用于正文（现代、清晰）
- **Oswald**: 西文 condensed，用于数字/标签（紧凑、冲击）
- **Inter**: 西文无衬线，用于英文内容（中性、现代）

**选择规则**:

| 内容类型 | 标题字体 | 正文字体 | 标签字体 |
|---------|---------|---------|---------|
| 纯中文 | Noto Serif SC | Noto Sans SC | Noto Sans SC |
| 中英混合 | Noto Serif SC | Inter | Noto Sans SC |
| 纯英文 | Oswald | Inter | Oswald |
| 数据密集 | Oswald | Inter | Oswald |

## 决策点 4: 色彩系统

**条件**: 确定信息卡的色彩方案

**默认方案**:
- **背景色**: #f5f3ed（米白色，纸质质感）
- **主文本**: #0a0a0a（近黑色）
- **正文**: #1a1a1a（深灰）
- **辅助**: #555555（中灰）
- **强调色**: #c9a87c（金棕色）

**自定义场景**:
- **科技主题**: 背景渐变、霓虹强调色
- **自然主题**: 大地色系、绿色调
- **商务主题**: 深蓝灰色、稳重视觉
- **时尚主题**: 高对比度、大胆配色

**决策依据**:
- 用户明确指定 → 使用用户配色
- 内容主题明确 → 使用主题相关配色
- 无特殊要求 → 使用默认方案

## 决策点 5: 响应式策略

**条件**: 确保在不同设备上的显示效果

**断点设置**:
- **移动端**: < 768px
- **平板**: 768px - 1024px
- **桌面**: > 1024px

**适配规则**:

| 元素 | 桌面 | 平板 | 移动 |
|-----|---------------------|-------------------|-------------------|
| 标题 | 72-140px | 56-84px | clamp(40px, 10vw, 72px) |
| 正文 | 18-20px | 17-19px | 16-18px |
| 外边距 | 50px | 40px | 30px |
| 栏数 | 1-3 栏 | 1-2 栏 | 1 栏 |

**关键原则**:
- 移动端正文字号 **不低于 16px**，推荐 **18px**
- 使用 `clamp()` 实现流畅缩放
- 高密度内容在移动端强制单栏

## 决策点 6: 视觉装饰程度

**条件**: 确定装饰元素的丰富程度

**装饰类型**:
- **噪点纹理**: 增加纸质质感
- **重型分割线**: 强化结构感
- **背景色块**: 区分内容区域
- **几何元素**: 现代感装饰

**决策矩阵**:

| 内容类型 | 装饰程度 | 使用元素 |
|---------|---------|---------|
| 极简内容 | 低 | 仅分割线 |
| 标准内容 | 中 | 分割线 + 背景色块 |
| 丰富内容 | 高 | 全部装饰元素 |

**原则**: 装饰服务于内容，不喧宾夺主
