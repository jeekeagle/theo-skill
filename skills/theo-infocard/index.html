核心能力

智能分析
自动评估信息密度（高/中/低），选择最佳布局策略
专业排版
完整字体、字号、行高、空间系统，确保视觉一致性
响应式布局
根据内容量自动选择"大字符主义"或"多栏网格"

生成包含完整 CSS 的 HTML 代码，可直接使用

设计原则

瑞士国际主义
  • 严谨的网格系统
  • 不对称布局平衡
  • 清晰的视觉层次
现代杂志质感
  • 超大视觉钩子
  • 重型分割线
  • 噪点纹理质感
核心策略
字号提升 + 紧凑排版 + 强化密度
字体系统
Noto Serif SC: 标题 900 Inter / Noto Sans SC: 正文 Oswald: 数字与强调

输出流程

Step 1
分析内容信息密度，确定布局策略
Step 2
生成完整 HTML 代码（含内联 CSS）
Step 3
自检移动端可读性（正文≥18px）
Step 4
保存 HTML 文件到本地
Step 5
使用 Chrome DevTools 生成 PNG 截图
何时使用
  • 内容需视觉化呈现
  • 创建社论质感卡片
  • 制作信息图谱
