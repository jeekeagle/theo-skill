# Theo's Agent Skills

个人定制的 Claude Code Agent Skills 集合。

## 可用 Skills

### 📝 [theo-post](./theo-post)

博客文章发布工具，将文章发布到 Theo 博客（AstroPaper 主题）。支持图片上传、格式化、frontmatter 生成和自动部署。

### 📚 [theo-paper](./theo-paper)

论文阅读辅助工具，帮助非学术背景人士理解学术论文，提取核心思想用于实际应用。

### 🎴 [theo-card](./theo-card)

内容铸造工具（铸），将内容转换为 PNG 视觉卡片。支持三种模具：长图（-l）、信息图（-i）、多卡（-c）。

### 🔄 [theo-paper-flow](./theo-paper-flow)

论文工作流，一条命令完成读论文 → 生成解读 → 铸成卡片。支持多篇并行处理。



## 使用方法

1. **命令方式**：输入斜杠命令，如 `/theo-post`
2. **自然语言**：直接说出触发词，如"发布文章"
